# 竞拍之王套狗计算器（和游泳统一颜色分级版）

一个使用 **Python + Tkinter** 编写的小工具，用于根据指定倍率计算“输入价格 ÷ 倍率”的结果，并按照数值大小使用不同颜色显示输入值和输出值。

## 功能说明

- 输入当前回合所出最高价
- 根据回合选择倍率：`2`、`1.6`、`1.3`、`1.1`
- 自动计算：`输入数字 ÷ 选择倍率`
- 输入值和结果值会根据数值范围自动变色：
  - **5位数（10000 ~ 99999）**：绿色
  - **十万级（100000 ~ 999999）**：蓝色
  - **百万级（1000000 ~ 9999999）**：黄色
  - **千万级及以上（10000000+）**：红色
  - **小于 10000**：黑色
- 支持图形界面操作
- 支持打包为 Windows `.exe`

## 示例

- 输入：`20000`
- 倍率：`2`
- 结果：`10000`

- 输入：`510000`
- 倍率：`2`
- 结果：`255000`
- 输入值和结果值都会显示为**蓝色**

## 项目文件说明

- `gui_calc_color.py`：主程序源码
- `除法计算器.spec`：PyInstaller 打包配置文件
- `dist/`：打包后的可执行文件输出目录（通常不提交到 Git）
- `build/`：PyInstaller 打包过程生成的临时目录（通常不提交到 Git）

## 运行方式

在项目目录中执行：

```bash
python gui_calc_color.py
```

如果你的环境使用指定解释器，也可以执行：

```bash
D:\app\AI\python.exe gui_calc_color.py
```

## 打包为 exe

先安装 PyInstaller：

```bash
pip install -U pyinstaller
```

然后在项目目录执行：

```bash
python -m PyInstaller --noconfirm --clean --onefile --windowed --name 除法计算器 gui_calc_color.py
```

打包完成后，可执行文件通常位于：

```bash
dist/除法计算器.exe
